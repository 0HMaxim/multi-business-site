# 🏢 Multi-Business SaaS Platform

A customizable web application that allows any business to run their own branded website and independently manage all content - without developer involvement.

🔗 **Live Demo:** https://0HMaxim.github.io/multi-business-site/

---

## 📌 About

This is a full-featured SaaS platform built for business entities. It includes a complete admin panel for content management, allowing non-technical users to manage everything on their own in **4 languages**.

---

## ✨ Features

- 🗂 **Admin Panel** - full content management system for all sections
- 👥 **Staff Profiles** - manage team members and highlight employees
- 🛠 **Services** - describe and update offered services
- 📝 **Blog** - publish news, updates, and articles
- 💰 **Pricing** - manage pricing tables and plans
- ❓ **FAQ** - add and edit frequently asked questions
- 🌍 **Multilingual Support** - content available in 4 languages
- 🌗 **Light / Dark Theme** - theme toggle for end users
- ➕ **Custom Pages** - add extra pages (e.g. job openings, employee of the month, news)
- ***  Customize header images and tab titles for each page
- 📷 **Photo Gallery** - upload and manage photos

---

## 🛠 Tech Stack

- **React** + **Vite** + **TypeScript**
- **Tailwind CSS**
- **Firebase Realtime Database**

---

## 💡 Goal

The platform is designed so that **any business** can run their own branded website and independently manage all content - staff, services, promotions, blog, FAQ, photos and Custom Page, HeaderImage and tab titels for each page - in 4 languages, without developer involvement.
